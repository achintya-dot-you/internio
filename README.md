## Run Locally

### Prerequisites: **Git**, **Node**

1. Clone the github repository using the Github CLI.

```sh
git clone https://github.com/achintya-dot-you/internio
```

2. Change to the project directory.

```sh
cd internio
```

3. Installation commands

```sh
npm install
```

4. Run

```sh
npm start
```
